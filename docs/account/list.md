# List accounts

List currently stored accounts

**URL** : `/accounts/`

**Method** : `GET`

## Success Response

**Condition** : If everything is OK

**Code** : `200`

**Content example**

```json
[
    {
        "balance": 1000.0,
        "currency": "EUR",
        "id": "c2642ae1-2b24-4551-96e5-65f07e8114fe"
    },
    {
        "balance": 1000.0,
        "currency": "EUR",
        "id": "0bdeaf7d-f3a3-47de-a44a-16fb721a93e3"
    }
]
```
